## Image Optimization Tips

**Size Guidelines:**
- **Recommended**: Under 50KB for cost efficiency
- **Maximum**: 100KB per transaction
- **Formats**: PNG (best quality) or JPEG (smaller files)
- **Resolution**: Optimize dimensions for your art style

**Creative Workflow:**
1. **Create artwork** using your preferred tools (Photoshop, GIMP, Procreate, etc.)
2. **Optimize file size** while maintaining visual quality
3. **Preview your stamp** using the web interface

5. **Share with community** on social platforms

## Cost & Network Considerations

**Fee Structure:**
- Network fees vary based on Bitcoin congestion (typically 15-50 sat/vB)
- Larger files cost more due to transaction size
- Create during off-peak hours for lower fees

**Best Practices:**
