# KEVIN Becomes the First SRC-20 Token

On May 3, 2023, at block 788,041, <EntityMention entity="arwyn">Arwyn</EntityMention> deployed <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> as the first-ever <EntityMention entity="src-20">SRC-20</EntityMention> token. It landed as Stamp #18,516, and it's the moment fungible tokens arrived on <EntityMention entity="bitcoin-stamps">Bitcoin Stamps</EntityMention>.

Up to that point, stamps were single pieces of permanent art. SRC-20 asked a different question: could you build a fungible-token standard on top of the same permanence guarantee? <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> is the answer, and it's still running.

## KEVIN before KEVIN the token

<EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> first showed up earlier, as Stamp #4258 at block 783,718 on April 3, 2023. That was the art. The token deploy at block 788,041 turned the character into the genesis SRC-20 asset, a fair-launch token with no presale and no insider allocation.

That ordering matters. The community recognized <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> as art first, then as the proof that the token standard worked. It wasn't a marketing launch. It was builders testing a new standard on real chain data.

## Setting the record straight

For a while some trackers listed a different token as the first SRC-20. That was wrong. A later CoinGecko correction confirmed <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> at block 788,041 as the genuine first SRC-20 deploy, not STAMP. Block heights don't lie, and the chain puts <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> first.

## Why it mattered

SRC-20 gave Bitcoin Stamps an account-based token model that lives in the UTXO set, so balances inherit the same permanence as the original art. Deploy, mint, and transfer operations all encode on Bitcoin. No sidechain, no bridge, no trusted host.

<EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> became the reference every later token points back to. It proved the standard, and it gave the community a mascot that means something: we're all <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention>.
