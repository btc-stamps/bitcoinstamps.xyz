# Bitcoin Stamps Kaynakları

<SmartStructuredData />

Bitcoin Stamps ekosistemi için kaynak dizini - ilk stampinizi oluşturmaktan uygulama geliştirmeye kadar.

## Resmi Platformlar

**[Stampchain.io →](https://stampchain.io)**
- Stamp'lere göz atın, SRC-20 tokenları takip edin, API erişimi
- 50.000'den fazla stamp kataloglandı
- Açık kaynaklı, topluluk tarafından sürdürülür

**[<EntityMention entity="kevin" variant="cultural">KEVIN </EntityMention> Ana Sayfası →](https://kevinstamp.com)**
- Köken hikayesi, topluluk güncellemeleri
- 2.300'den fazla KEVIN token sahibi

**[API Dokümantasyonu →](https://stampchain.io/docs)**
- REST API, webhook entegrasyonu
- Geliştiriciler için teknik dokümantasyon

## Sanatçı Kaynakları

**Oluşturma Araçları:**
- Piksel sanat editörleri ve stamp doğrulama
- PNG, GIF, SVG desteği (<100KB için optimize edin)
- Gerçek zamanlı önizleme ve maliyet tahmini
- Koleksiyonlar için toplu oluşturma

**SRC-20 Token Oluşturma:**
- stampchain.io'ya entegre
- Adil başlatma (ön madencilik yok)
- Topluluk mentorluğu mevcut

**Öğrenme Materyalleri:**
- Piksel sanat eğitimleri
- Renk paleti optimizasyonu
- Bitcoin depolama temelleri
- Topluluk galeri vitrini
