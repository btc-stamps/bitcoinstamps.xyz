## 交易成本

比特币印章直接在比特币上存储数据，这意味着交易成本根据网络条件而变化。

### 对于艺术家
- 小型印章通常花费 3-6 美元的比特币手续费
- 较大的印章根据数据大小可能花费 10-20 美元以上
- 成本随比特币网络拥堵情况变化

### 成本估算
```typescript
function estimateStampCost(dataSize: number, feeRate: number): number {
  const overhead = 200; // 字节
  return (dataSize + overhead) * feeRate;
}
```
