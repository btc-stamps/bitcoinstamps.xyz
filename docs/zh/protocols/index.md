# Bitcoin Stamps 协议

<SmartStructuredData />

Bitcoin Stamps 使用多个元协议来创建比特币上的不同类型数字资产。每个协议都有独特的用途，从创建数字艺术到复杂的代币系统。

## 主要协议

### 🪙 [SRC-20 代币](/en/protocols/src-20) *（英文版）*
**比特币上的可替代代币**  
创建和交易像 <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> 这样的代币，这是第一个 SRC-20 代币，为社区驱动的开发铺平了道路。

- 代币发行
- 账户间转账
- 燃烧机制
- 社区治理

### 🏷️ [SRC-101 名称](/en/protocols/src-101) *（英文版）*
**人类可读的命名系统**  
为比特币地址和资源创建易记的名称，让生态系统对用户更加友好。

- 名称注册
- 地址解析
- 命名空间元数据
- 续期管理

### 🎨 [SRC-721 NFT](/en/protocols/src-721) *（英文版）*
**高级非同质化代币**  
创建具有链上功能、递归艺术和交互特性的复杂 NFT。

- 独特的 NFT
- 递归功能
- 链上元数据
- 交互特性

### ⚡ [OLGA 压缩](/en/protocols/olga) *（英文版）*
**所有协议的优化**  
通过智能压缩编码降低交易成本并提高所有 Bitcoin Stamps 协议的效率。

- 数据压缩
- 成本降低
- 效率优化
- 通用兼容性

## 系统架构

所有 Bitcoin Stamps 协议共享：

- **链上存储**：数据直接存储在比特币交易中
- **基于账户的模型**：简化的所有权管理
- **Counterparty 共识**：成熟的信任基础设施
- **社区索引**：分布式和弹性的索引系统

## 协议入门

根据你的用例选择路径：

- **创作者**：从 [SRC-721 NFT](/en/protocols/src-721) 开始创作数字艺术 *（英文版）*
- **代币开发者**：探索 [SRC-20](/en/protocols/src-20) 创建可替代代币 *（英文版）*
- **基础设施服务**：实现 [SRC-101](/en/protocols/src-101) 提供友好的地址服务 *（英文版）*
- **成本优化**：使用 [OLGA](/en/protocols/olga) 提高效率 *（英文版）*
