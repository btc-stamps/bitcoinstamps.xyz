## 创建前检查清单

1. **定义**您的艺术愿景和文化贡献
2. 将图片**优化**到 50KB 以下以获得合理费用（最大 100KB）
3. **检查**网络费用并相应制定预算
4. **验证**连接钱包中的比特币余额

## 图片优化技巧

**尺寸指南：**
- **推荐**：50KB 以下以获得成本效益
- **最大**：每笔交易 100KB
- **格式**：PNG（最佳质量）或 JPEG（较小文件）
- **分辨率**：为您的艺术风格优化尺寸

**创意工作流程：**
1. 使用您喜欢的工具**创建艺术品**（Photoshop、GIMP、Procreate 等）
2. 在保持视觉质量的同时**优化文件大小**
3. 使用 Web 界面**预览您的邮票**
4. 通过 [stampchain.io](https://stampchain.io) **创建邮票**
5. 在社交平台上**与社区分享**

## 成本和网络考虑

**费用结构：**
- 网络费用根据比特币拥堵情况变化（通常为 15-50 聪/vB）
- 较大的文件由于交易大小而成本更高
- 在非高峰时段创建以获得较低费用

**最佳实践：**
- 先用小金额测试
- 考虑社区影响——创建有意义的内容
- 长期考虑——邮票在比特币存在期间永远存在
- 尊重 KEVIN 建立的社区价值观
