# KEVIN 成为第一个 SRC-20 代币

2023年5月3日，在区块 788,041，<EntityMention entity="arwyn">Arwyn</EntityMention> 将 <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> 部署为有史以来第一个 <EntityMention entity="src-20">SRC-20</EntityMention> 代币。它以 Stamp #18,516 的身份落地，也标志着同质化代币在 <EntityMention entity="bitcoin-stamps">Bitcoin Stamps</EntityMention> 上的到来。

在那之前，stamps 都是单件的永久艺术品。SRC-20 提出了一个不同的问题：能否在同一套永久性保证之上构建一个同质化代币标准？<EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> 就是答案，而且它至今仍在运行。

## 代币 KEVIN 之前的 KEVIN

<EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> 其实更早就出现过，那是 2023年4月3日、在区块 783,718 上的 Stamp #4258。那是艺术。区块 788,041 上的代币部署，则把这个角色变成了创世级的 SRC-20 资产——一个没有预售、没有内部人分配的公平启动代币。

这个先后顺序很重要。社区先把 <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> 认作艺术，然后才把它认作代币标准可行的证明。它不是一次营销启动，而是建设者们在真实链上数据上测试一个新标准。

## 澄清事实

有一段时间，一些追踪工具把另一个代币列为第一个 SRC-20。那是错的。CoinGecko 后来的一次更正确认，区块 788,041 上的 <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> 才是真正的第一个 SRC-20 部署，而不是 STAMP。区块高度不会说谎，链把 <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> 排在首位。

## 它为什么重要

SRC-20 为 Bitcoin Stamps 带来了一个存活于 UTXO 集中的、基于账户的代币模型，因此余额继承了与原始艺术相同的永久性。部署、铸造和转移操作全部编码在比特币上。没有侧链，没有跨链桥，没有需要信任的主机。

<EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> 成了此后每一个代币回溯参照的基准。它证明了这个标准，也给了社区一个意义深远的吉祥物：我们都是 <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention>。
