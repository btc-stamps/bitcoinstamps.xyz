# Ressources Bitcoin Stamps

<SmartStructuredData />

Répertoire de ressources pour l'écosystème Bitcoin Stamps - de la création de votre premier stamp à la construction d'applications.

## Plateformes Officielles

**[Stampchain.io →](https://stampchain.io)**
- Parcourir les stamps, suivre les jetons SRC-20, accès API
- Plus de 50 000 stamps catalogués
- Open-source, maintenu par la communauté

**[Page d'accueil <EntityMention entity="kevin" variant="cultural">KEVIN </EntityMention> →](https://kevinstamp.com)**
- Histoire d'origine, mises à jour communautaires
- Plus de 2 300 détenteurs du jeton KEVIN

**[Documentation API →](https://stampchain.io/docs)**
- API REST, intégration webhook
- Documentation technique pour les développeurs

## Ressources pour Artistes

**Outils de Création :**
- Éditeurs d'art pixel et validation de stamp
- Support PNG, GIF, SVG (optimiser pour <100KB)
- Prévisualisation en temps réel et estimation des coûts
- Création en lot pour les collections

**Création de Jetons SRC-20 :**
- Intégré dans stampchain.io
- Lancement équitable (pas de pré-minage)
- Mentorat communautaire disponible

**Matériaux d'Apprentissage :**
- Tutoriels d'art pixel
- Optimisation de palette de couleurs
- Bases du stockage Bitcoin
- Galerie vitrine communautaire
