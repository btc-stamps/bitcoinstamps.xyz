# Qu'est-ce que les Bitcoin Stamps ? Des Actifs Numériques Renforçant le Réseau

Les Bitcoin Stamps sont des **actifs numériques permanents** stockés directement sur la blockchain Bitcoin. Contrairement aux NFT traditionnels qui dépendent de serveurs externes, les Bitcoin Stamps intègrent toutes leurs données directement dans les transactions Bitcoin, les rendant véritablement immuables et indépendants.

## Ce qui rend les Bitcoin Stamps spéciaux

- **Permanence véritable** : Les données vivent pour toujours sur Bitcoin, le réseau le plus sécurisé au monde
- **Aucune dépendance** : Ne dépend pas de sites web, serveurs ou stockage tiers
- **Renforcement du réseau** : Chaque transaction de stamp soutient les mineurs Bitcoin par les frais
- **Valeurs communautaires** : Guidés par les principes de <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> d'équité et de créativité authentique
- **Innovation technique** : Métaprotocoles avancés construits sur la fondation solide de Bitcoin

## Architecture technique : une permanence fondée sur les UTXO

Les Bitcoin Stamps adoptent une approche fondamentalement différente du stockage des données par rapport aux autres actifs numériques basés sur Bitcoin :

### Modèle de stockage P2WSH
Les Bitcoin Stamps stockent les données à l'aide de l'**encodage P2WSH (Pay-to-Witness-Script-Hash)** directement dans l'ensemble UTXO de Bitcoin :

- **Stockage universel sur les nœuds** : TOUS les nœuds complets Bitcoin stockent les données des stamps de façon permanente
- **Critique pour le consensus** : Les données font partie des exigences fondamentales de validation de Bitcoin
- **Impossibles à élaguer** : Elles ne peuvent être supprimées ni élaguées de la blockchain
- **Économie intégrée** : La création de stamps soutient intrinsèquement les mineurs Bitcoin

### Pourquoi cela compte pour la permanence

Contrairement aux approches fondées sur les données de témoin (witness data), les données des Bitcoin Stamps sont :
- **Nécessaires au fonctionnement de Bitcoin** : Les nœuds doivent stocker les données UTXO pour valider les transactions
- **Protégées par le consensus** : Elles font partie du modèle économique et de sécurité fondamental de Bitcoin
- **Pérennes** : Elles survivent aussi longtemps que le réseau Bitcoin existe
- **Auto-validantes** : Elles ne requièrent aucun service ni infrastructure externe

## Comment les Bitcoin Stamps renforcent le réseau

Chaque transaction Bitcoin Stamp :
- **Soutient les mineurs** : Les frais de transaction fournissent des revenus aux mineurs Bitcoin
- **Augmente la sécurité** : Plus de transactions renforcent le mécanisme de consensus Bitcoin
- **Crée de l'utilité** : Cas d'usage réels au-delà du simple transfert de valeur
- **Verrouille de la valeur** : Certains types de stamps verrouillent définitivement du Bitcoin dans des sorties spéciales
