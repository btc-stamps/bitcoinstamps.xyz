# Qu'est-ce que les Bitcoin Stamps ? Des Actifs Numériques Renforçant le Réseau

<SmartStructuredData />

Les Bitcoin Stamps sont des **actifs numériques permanents** stockés directement sur la blockchain Bitcoin. Contrairement aux NFT traditionnels qui dépendent de serveurs externes, les Bitcoin Stamps intègrent toutes leurs données directement dans les transactions Bitcoin, les rendant véritablement immutables et indépendants.

## Ce qui rend les Bitcoin Stamps spéciaux

- **Permanence véritable** : Les données vivent pour toujours sur Bitcoin, le réseau le plus sécurisé au monde
- **Aucune dépendance** : Ne dépend pas de sites web, serveurs ou stockage tiers
- **Renforcement du réseau** : Chaque transaction de stamp soutient les mineurs Bitcoin par les frais
- **Valeurs communautaires** : Guidés par les principes de <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> d'équité et de créativité authentique
- **Innovation technique** : Métaprotocoles avancés construits sur la fondation solide de Bitcoin

## Protocoles Bitcoin Stamps

### Jetons SRC-20
Jetons fongibles sur Bitcoin via <EntityMention entity="src-20" variant="technical">SRC-20</EntityMention>. <EntityMention entity="kevin" variant="cultural">KEVIN</EntityMention> était le premier jeton SRC-20, démontrant le développement communautaire et les principes de lancement équitable.

### Noms SRC-101  
Noms lisibles par l'homme pour les adresses Bitcoin et ressources utilisant <EntityMention entity="src-101" variant="technical">SRC-101</EntityMention>, rendant l'écosystème plus accessible.

### NFT SRC-721
Jetons non fongibles avancés avec des capacités récursives et des fonctionnalités on-chain complexes grâce à <EntityMention entity="src-721" variant="technical">SRC-721</EntityMention>.

### Encodage OLGA
Optimisation d'encodage P2WSH qui réduit les coûts sur tous les protocoles Bitcoin Stamps via <EntityMention entity="olga" variant="technical">OLGA</EntityMention>.

## Comment les Bitcoin Stamps renforcent le réseau

Chaque transaction Bitcoin Stamp :
- **Soutient les mineurs** : Les frais de transaction fournissent des revenus aux mineurs Bitcoin
- **Augmente la sécurité** : Plus de transactions renforcent le mécanisme de consensus Bitcoin
- **Crée de l'utilité** : Cas d'usage réels au-delà du simple transfert de valeur
- **Verrouille de la valeur** : Certains types de stamps verrouillent définitivement du Bitcoin dans des sorties spéciales
